JFSplit <http://jfsplit.sourceforge.net> is a free file splitter and joiner with a number options to split and join your files efficiently. Cross-platform tool programmed in JAVA and requires Java Runtime Environment to use this tool.

This tool uses MiGLayout <http://www.miglayout.com/> which is an Open Source Java Layout Manager used to layout GUI components.
